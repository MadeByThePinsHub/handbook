    README

    Welcome to the Pins team's official English Handbook! This is where the whole handbook lives and resides, mostly adopted from GitLab Handbook.

    Heads up, the handbook is still work-in-progress. Feedback is appericated! :love:

    You'll seeing this because most of the deployments are either from the develop branch or from an branch we're working. For any feedback about this preview, use our issue tracker in GitLab.com.
